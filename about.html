About

Built for simple squads.

Near2Squad is a lightweight concept for gamers who want fast rooms, clear states, and accessible controls without copied image assets.

Original visual system

The interface uses CSS gradients, glass panels, and text-only marks instead of external images.

Accessible by default

Controls include visible focus styles, readable contrast, and mobile-first layouts.

Near2Squad uses original UI only. No Google images or copyrighted assets are included.
